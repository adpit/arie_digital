    • Unique theme features

      • Dedicated search plank
      • Taxonomy support
      • Configur'ble topbar buttons

      • Configur'ble shortcut links

      • Image effects like siz'n, shadow, border an' alignment
      • Image lightbox
    • Shorrrtcodes galore

      • Configur'ble buttons
      • List child planks
      • Expand areas t' reveal rrrambl'n
      • Font Awesome ay'cons
